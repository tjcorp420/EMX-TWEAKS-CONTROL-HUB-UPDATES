## EMX Tweaks Control Hub v1.0.2

This release changes updates from opening a setup download link to an in-app install and restart flow.

- Adds a glowing Install & Restart update button when an update is available.
- Adds a branded animated update overlay.
- Downloads latest app files from GitHub in the background.
- Verifies the update package SHA-256 before installing.
- Stages the files, closes the app, replaces the installed app files, and restarts automatically.
